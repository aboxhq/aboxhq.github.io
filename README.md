# Public Assets

This directory contains static assets that are served directly without processing.

## Files

- `favicon.svg` - Website favicon (SVG format)
- `CNAME` - Custom domain configuration for GitHub Pages
- `ffmpeg.js` - FFmpeg JavaScript library for media processing
- `wasm/` - WebAssembly modules for high-performance processing
- `sample-images/` - Sample images for testing

## Note

Files in this directory are copied as-is to the build output.
